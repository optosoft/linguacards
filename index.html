❓ Help & Guide

🌍
Welcome to LinguaCards!
LinguaCards lets you create flashcard decks for any language pair, study with flip cards, test yourself with quizzes, and generate decks instantly with AI.
1
Create a deck — tap + New Deck, enter a name, choose your Word Language (front of card) and Definition Language (back), then pick a colour.
2
Add cards — open your deck and tap + Add Card. Fill in the front, back, an optional hint, and an optional image.
3
Study or Quiz — tap 📖 Study to flip through cards, or ⚡ Quiz for multiple-choice testing.
4
Or generate instantly — tap 🤖 AI Generate, enter a topic and language pair, and a full deck is created in seconds.
📖
Study Mode
Flip through cards one by one. Tap the card or the Flip button to reveal the answer.
↩ FlipReveal the back of the card
Next →Skip to the next card without scoring
✓ Got ItMark correct and move on
✗ Try AgainMark incorrect and move on
🔊Speak the word on that card face
⚡
Quiz Mode
Four multiple-choice options are shown for each card. The correct answer is highlighted green after you pick. Your score is shown at the end.
📋
Card Order
In the deck view, choose how cards are ordered before studying: Original (insertion order), A–Z (alphabetical by front), or Shuffle (random each session).
✏️
Manual Deck
Tap + New Deck, set a name, word language (front) and definition language (back). Then add cards one by one with + Add Card. Each card supports a front, back, optional hint, and optional photo.
🤖
AI Generate
Tap 🤖 AI Generate and enter a topic (e.g. "Food & Drinks"), a word language, and a definition language. Claude generates a ready-to-study deck instantly. Requires an Anthropic API key when running outside Claude.ai.
📄
Import from CSV
Tap 📄 From CSV and upload a .csv file. Map the columns to Front, Back, and Hint, set the deck name and languages, then tap Create Deck. Rows with empty front or back are skipped automatically.
💡 CSV tip: Name your columns front, back, and hint and they will be auto-detected.
⌨️
Phonetic Keyboard
When adding or editing a card, an ⌨️ Keyboard button appears next to any field whose language uses a non-Latin script. Tap it to open the on-screen keyboard.
🇯🇵 JapaneseHiragana, Katakana, and a Rōmaji → かな converter
🇰🇷 KoreanHangul with live syllable composition (ㄱ+ㅏ → 가)
🇸🇦 ArabicFull alphabet + diacritics, right-to-left layout
🇷🇺 RussianCyrillic upper and lower case
🇬🇷 GreekGreek alphabet with accented vowels
🇮🇳 HindiDevanagari consonants, vowels, and matras
🇹🇭 ThaiThai consonants, vowels, and tone marks
🇮🇱 HebrewHebrew letters + vowel points, right-to-left
🇨🇳 ChinesePinyin tone marks (ā á ǎ à)
🇺🇦 UkrainianUkrainian Cyrillic alphabet
💡 Every key shows the native character (large) and its Latin phonetic hint (small), so you can find characters by sound.
🔊
Text-to-Speech
LinguaCards uses your device's built-in speech engine to speak cards. Each side is spoken in its own native language accent — English in an English voice, Japanese in a Japanese voice, and so on.
🔊 OnMaster switch — enables all audio
🔇 OffDisables all audio including manual taps
▶ ManualAudio only when you tap the 🔊 button on a card
⚡ AutoAudio plays automatically on each card and flip
💡 Voice quality depends on which language packs are installed on your device. On iPhone and iPad, installing a language in Settings → General → Language & Region gives the best results.
⬇️
Export
Tap 📤 Export inside a deck to save that deck as a JSON file, or use Import/Export in the top bar to export all decks at once.
⬆️
Import
Use Import/Export → Import Decks from File to load a previously exported JSON file. If a deck with the same ID already exists it will be updated; otherwise it is added as a new deck.
☁️
Cloud Save
All decks are automatically saved to cloud storage when you make changes. Your data persists across sessions without any extra steps.
💡 Images on cards are stored as base64 inside the JSON, so exported files with many photos can be large.
