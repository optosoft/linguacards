❓ Help & Guide

🌍
Welcome to LinguaCards!
Create flashcard decks for any language pair, study with flip cards, test yourself with quizzes, and generate decks instantly with AI.
1
Create a deck — tap + New Deck, set a name and both languages.
2
Add cards — tap + Add Card. Each card has a front, back, optional hint, and optional photo.
3
Study or Quiz — tap 📖 Study to flip through cards, or ⚡ Quiz for multiple-choice.
4
Track progress — the mastery bar in each deck shows your learning progress. Quiz results update it automatically.
📖
Study Mode
Flip through cards one by one. Use the direction toggle to switch which side appears first.
🔤 Word firstShows the word (front), reveals definition on flip
📖 Def firstShows the definition (back), reveals word on flip
← PrevGo back to the previous card
↩ FlipReveal the other side
Next →Skip without scoring
✓ Got It / ✗ Try AgainScore and advance
⚡
Quiz Mode
Multiple-choice quiz with smart card weighting — cards you struggle with appear more often.
🔤 Word→DefShows word, pick the correct definition
📖 Def→WordShows definition, pick the correct word
Questions: 5/10/20/AllHow many cards to quiz per session
⚡ Weak cardsCards with low streak are weighted higher
📊
Progress Tracking
Each correct quiz answer increases a card's streak. 3 correct answers in a row = mastered (green). Getting it wrong resets the streak. Use ↺ Reset Progress in the deck view to start over.
✏️
Manual Deck
Tap + New Deck, then add cards one by one.
🤖
AI Generate
Enter a topic and language pair. Claude generates a full deck instantly. Requires an API key outside Claude.ai.
📄
CSV Import
Upload a .csv file and map columns to Front / Back / Hint.
💡 Name your CSV columns front, back, hint for auto-detection.
⌨️
Phonetic Keyboard
An ⌨️ button appears next to card fields when the language has a non-Latin script. Every key shows the character plus its phonetic hint.
🇯🇵 JapaneseHiragana, Katakana, Rōmaji→かな
🇰🇷 KoreanHangul with live syllable composition
🇸🇦 Arabic · 🇮🇱 HebrewRTL keyboards with diacritics
🇷🇺 🇬🇷 🇮🇳 🇹🇭 🇨🇳 🇺🇦Russian, Greek, Hindi, Thai, Pinyin, Ukrainian
🔊
Text-to-Speech
Each card side speaks in its own native language accent.
🔊 OnEnables all audio
🔇 OffDisables all audio
▶ ManualPlay only when you tap 🔊
⚡ AutoPlays automatically on each card
💡 On iPhone/iPad, install language packs in Settings → General → Language & Region for best quality.
